Open and portable plotting

OxyPlot is an open source and cross-platform plotting library for .NET.

The code is licensed under the very liberal MIT license. This means that the library can be used in commercial applications, and modified as much as you want. But remember there is no warranty or support included...

What does it look like?

The same plotting code can be used on different user interface platforms, and the plots can be exported to file formats like png, pdf and svg.

Getting started

It should not take much time to get a plot running in your application. Use the NuGet package manager to add the OxyPlot references. Then add a "Plot" control to your view and create a "PlotModel" that is bound to this control.
