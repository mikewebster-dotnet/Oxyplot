OxyPlot
==============================================================================
OxyPlot is a .NET library for making XY line plots. The focus is on simplicity 
and performance. The library contains custom controls for WPF, Silverlight and 
Windows Forms. The plots can be exported to SVG, PDF and PNG.

License:       Ms-PL
Project page:  http://oxyplot.codeplex.com
SVN:           https://oxyplot.svn.codeplex.com/svn

Build requirements
- Microsoft .NET 4.0 Client Profile
- Visual Studio 2010
